Calculadora en HTML con Bootstrap
Este proyecto es una calculadora simple implementada en HTML utilizando Bootstrap para la interfaz de usuario.

Descripción
La calculadora ofrece una interfaz de usuario básica con botones para los números del 0 al 9, así como operadores de suma, resta, multiplicación y división. También incluye botones para borrar la pantalla de visualización y calcular el resultado de la expresión matemática ingresada.

Requisitos
Para ejecutar esta calculadora, asegúrate de tener una conexión a Internet activa para cargar las bibliotecas Bootstrap desde la web. No se requiere ninguna instalación adicional.

Uso
Abre el archivo HTML en tu navegador web.
Utiliza los botones numéricos para ingresar los números que desees en la pantalla de visualización.
Utiliza los botones de operadores (+, -, *, /) para realizar operaciones matemáticas.
Puedes borrar la pantalla de visualización presionando el botón "C".
Para calcular el resultado, presiona el botón "=".
Personalización
Si deseas personalizar la calculadora o hacer cambios en su apariencia, puedes modificar el código HTML y CSS según tus necesidades. El archivo CSS personalizado se encuentra en la sección <style> del archivo HTML.

Créditos
Este proyecto utiliza la biblioteca Bootstrap para la interfaz de usuario.
